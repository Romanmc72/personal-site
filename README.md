# Personal Site

This is the code for my personal website hosted for free on cloudflare. Deployments are through wrangler on changes pushed to github.

## What's in here

It started with my resume, but it will probably hold licensing and legal pages like terms and conditions for various apps I build over time. Just a really quick and dirty way to publish web content with little to no effort on my end.

## Workflow

1. prompt AI to make a page
1. look at it to make sure it is good (if not repeat step 1)
1. git add . && git commit -m 'Added another page' && git push
1. 💰 profit 💰
